Login

Purpose

The Login page is the secure entry point into the True Care System platform. Every authenticated session begins here.

Prerequisites

  • Valid user account
  • Assigned role and permissions
  • Supported browser
  • Internet connection

Login Screen

The following screen is used by all authorized users.

Login Fields

FieldDescription
Email / UsernameUser account assigned by the organization.
PasswordSecure password associated with the account.
Sign InAuthenticates the user and opens the assigned dashboard.
Forgot PasswordStarts the password reset process.

Login Process

  1. Open the Login page.
  2. Enter Email or Username.
  3. Enter Password.
  4. Click Sign In.
  5. System authenticates credentials.
  6. User is redirected to the appropriate dashboard.

Successful Login

  • Dashboard is displayed.
  • Menus are loaded according to the assigned role.
  • A secure session is created.
  • The login event is recorded in Audit Logs.

Security

  • HTTPS encrypted connection.
  • Role-based access control.
  • Session timeout protection.
  • Future MFA support.
  • Failed login monitoring.

HIPAA Considerations

  • Never share credentials.
  • Always log out when leaving the workstation.
  • Do not save passwords on shared computers.
  • Protect PHI from unauthorized viewing.

Common Errors

  • Invalid username or password.
  • Account disabled.
  • Session expired.
  • No internet connection.

Troubleshooting

  • Verify Caps Lock.
  • Confirm internet connectivity.
  • Use Forgot Password if needed.
  • Contact your Provider Administrator if access remains unavailable.

FAQ

Can I use the same account on multiple computers?
Yes, provided your organization permits it and each session complies with security policies.

What happens after multiple failed login attempts?
The organization may enforce security controls such as temporary lockout.
