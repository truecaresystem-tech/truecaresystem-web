Login

Purpose

The Login page is the secure authentication gateway to the True Care System platform. All authorized users must successfully sign in before accessing protected information, role-based modules, and operational features.

Prerequisites

  • Active True Care System account.
  • User role assigned by Provider Administrator.
  • Supported web browser.
  • Internet connection.

Login Process

  1. Open the Login page.
  2. Enter your Email or Username.
  3. Enter your Password.
  4. Click Sign In.
  5. The system validates your credentials.
  6. The appropriate dashboard is displayed.

Successful Login

  • Dashboard loads successfully.
  • Menus are displayed based on the assigned role.
  • A secure authenticated session is established.
  • Authentication activity is recorded.

Role-Based Access

Each user only has access to modules and functions granted by their assigned role. Examples include Provider Administrator, Supervisor, DSP, Nurse, Billing, Payroll, Office Staff, Compliance, and Support.

Security

  • HTTPS encrypted communication.
  • Role-based authorization.
  • Secure session management.
  • Automatic session timeout.
  • Failed login monitoring.
  • Future Multi-Factor Authentication (MFA) support.

HIPAA Considerations

  • Never share your credentials.
  • Always log out before leaving your workstation.
  • Do not save passwords on shared computers.
  • Protect all PHI from unauthorized access.

Audit Logging

The system may record authentication events including:

  • User ID
  • Date and Time
  • IP Address
  • Browser
  • Device
  • Login Result (Success / Failure)

Best Practices

  • Bookmark the official login page.
  • Use a strong password.
  • Lock your workstation when away.
  • Never share accounts.
  • Sign out after each session.

Troubleshooting

  • Verify Email and Password.
  • Check Caps Lock.
  • Refresh the page.
  • Verify internet connectivity.
  • Use Forgot Password if necessary.
  • Contact your Provider Administrator if the issue persists.

FAQ

Can I sign in from multiple devices?
Yes, if permitted by your organization.

Why don't I see all menus?
Menus are displayed according to your assigned role.

Are login events recorded?
Yes. Authentication events may be logged for security and HIPAA compliance.
