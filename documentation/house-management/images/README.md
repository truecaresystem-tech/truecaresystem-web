# House Management Screenshot Library

Store screenshots for the **House Management** documentation in this folder.

Naming format:

`HOU-001-description.jpg`

Requirements:

- Use demo or training data only.
- Do not upload real PHI.
- Do not expose passwords, API credentials, SSNs or full Medicaid IDs.
- Capture desktop screens at 100% browser zoom unless otherwise documented.
- Keep screenshot IDs stable when replacing an outdated image.
